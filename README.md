# CSV Data Processor & Manager

## Overview
A Python utility for parsing, manipulating, and querying structured CSV data using Object-Oriented Programming and file-based data persistence.

## Technical Methodology
* **Object-Oriented Design**: Implemented a `Shoe` class to manage attributes and encapsulate data[cite: 4].
* **File I/O Operations**: Developed custom read/write logic to handle persistent data storage[cite: 4].
* **Data Visualization**: Integrated the `tabulate` library to dynamically format console output[cite: 4].
* **Algorithmic Logic**: Implemented min/max search logic to identify stock levels and calculate aggregate financial values[cite: 4].
